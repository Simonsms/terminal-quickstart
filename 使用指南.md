# Terminal QuickStart 使用指南

## 🚀 项目介绍

Terminal QuickStart 是一个基于 Vue 3 + Tauri 开发的桌面应用，用于快速管理和启动终端脚本。

### ✨ 核心功能

- 📝 **脚本管理** - 添加、编辑、删除脚本配置
- 🚀 **一键启动** - 在指定工作目录快速执行命令
- 💾 **配置持久化** - 自动保存所有脚本配置
- 📁 **目录选择器** - 图形化选择工作目录
- 🌙 **暗色主题** - 舒适的深色界面设计

---

## 📦 安装与运行

### 开发环境

#### 前置要求
- Node.js >= 16
- Rust >= 1.70
- Cargo

#### 安装依赖
```bash
cd terminal-quickstart
npm install
```

#### 启动开发服务器
```bash
npm run tauri dev
```

应用将自动打开，Vite 前端运行在 `http://localhost:1420`

---

### 生产构建

#### 打包应用
```bash
npm run tauri build
```

生成的安装包位于：
- Windows: `src-tauri/target/release/bundle/msi/`
- 可执行文件: `src-tauri/target/release/terminal-quickstart.exe`

## 📖 使用说明

### 1. 添加脚本

1. 点击右上角 **「新建脚本」** 按钮
2. 填写脚本信息：
   - **脚本名称**: 例如 "启动前端项目"
   - **工作目录**: 点击「浏览」选择项目目录，或手动输入路径
   - **启动命令**: 例如 `npm run dev`
   - **描述信息**: 可选，添加脚本说明
3. 点击 **「保存」** 完成添加

### 2. 启动脚本

- 找到要启动的脚本卡片
- 点击 **「启动」** 按钮
- 系统会在新的 CMD 窗口中执行命令

### 3. 编辑脚本

- 点击脚本卡片上的 **「编辑」** 图标按钮
- 修改脚本信息
- 点击 **「保存」** 应用更改

### 4. 删除脚本

- 点击脚本卡片上的 **「删除」** 图标按钮（红色）
- 确认删除操作

## 🎨 界面说明

### 左侧导航栏
- **脚本管理** - 脚本列表和管理页面
- **设置** - 应用设置（开发中）

### 脚本卡片
每个脚本卡片显示：
- 📁 图标和脚本名称
- 📂 工作目录路径
- 💻 执行命令预览
- 📝 描述信息（如果有）
- 🔘 操作按钮（启动、编辑、删除）

### 空状态
没有脚本时显示引导提示，点击「新建脚本」开始使用

## 📁 配置文件位置

应用配置自动保存在：

```
Windows: C:\Users\{用户名}\AppData\Roaming\com.tauri.terminal-quickstart\config.json
```

配置文件格式：
```json
{
  "theme": "dark",
  "version": "1.0.0",
  "scripts": [
    {
      "id": "script-1234567890-abc123",
      "name": "示例脚本",
      "workingDir": "C:\\projects\\my-project",
      "command": "npm run dev",
      "description": "开发服务器",
      "createdAt": 1701234567890,
      "updatedAt": 1701234567890
    }
  ]
}
```

## 🔧 故障排查

### 问题 1：脚本启动失败

**可能原因**:
- 工作目录路径不正确
- 命令拼写错误
- 缺少必要的环境依赖

**解决方案**:
1. 检查工作目录是否存在
2. 在终端手动测试命令
3. 确认环境变量配置正确

### 问题 2：找不到配置文件

**解决方案**:
- 首次运行时会自动创建
- 手动删除配置文件可重置应用

### 问题 3：应用无法启动

**检查项**:
- Node.js 版本 >= 16
- Rust 已正确安装
- 端口 1420 未被占用
